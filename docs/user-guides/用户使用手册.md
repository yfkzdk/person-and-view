## 🔧 常见问题解决

### ❌ 端口8000已被占用
**原因：** 其他程序占用了8000端口
**解决：**
```powershell
# 查看占用进程
netstat -ano | findstr :8000

# 结束进程（替换PID为实际进程ID）
taskkill /PID <进程ID> /F

# 或修改端口（编辑run_server.py最后一行）
uvicorn.run(app, host="0.0.0.0", port=8001)
```

### ❌ 前端无法连接后端
**检查清单：**
1. ✅ 后端是否在运行？
2. ✅ 地址是否正确？(http://localhost:8000)
3. ✅ 浏览器控制台有无错误？
4. ✅ 防火墙是否阻止？

### ❌ AI没有回复
**可能原因：**
1. API密钥未配置 → 配置`.env`文件
2. API余额不足 → 检查账户余额
3. 网络问题 → 检查网络连接

### ❌ 语音功能不工作
**解决方法：**
1. 使用Chrome或Edge浏览器
2. 允许浏览器访问麦克风权限
3. 生产环境需要HTTPS（开发环境可用HTTP）
